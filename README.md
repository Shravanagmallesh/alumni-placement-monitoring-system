# Digital Platform for Training and Placement Cell

This is a comprehensive digital solution designed to modernize and simplify the training and placement processes for students, recruiters, and placement coordinators at our institution. By digitizing traditional methods, we aim to make job searching, application tracking, and skill development more efficient and accessible.

## Project Overview

Our platform serves as a bridge between students seeking opportunities and recruiters looking for talent. It includes features for student registration, job posting, application management, and resources for skill enhancement. We've incorporated automation for notifications and data security to ensure a smooth experience for all users.

### Key Benefits
- Streamlined job applications and progress tracking
- Automated alerts for new opportunities
- Access to aptitude tests and coding resources
- Networking opportunities with alumni
- Robust security measures for user data

## Technology Stack
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Backend**: Flask (Python)
- **Database**: MySQL
- **Services**: Gmail API for email notifications

## Features
- **Student Dashboard**: Profile management, application tracking, job alerts
- **Recruiter Panel**: Job posting and candidate review
- **Notifications**: Real-time updates via email
- **Skill Development**: Practice tests and resources
- **Alumni Network**: Connect with former students for guidance
- **Authentication**: Secure login and data protection

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/your-repo-url.git
   cd Digital-Platform-for-Training-and-Placement-Cell
   ```

2. Install the required dependencies:
   ```
   pip install flask pandas openpyxl werkzeug
   ```

3. Start the Flask server:
   ```
   python app.py
   ```

4. Open your browser and go to `http://localhost:5000`
